# AndanteTribe

AndanteTribe はゲーム制作を行うクリエイティブグループです。現在の開発では Unity と .NET (C#) を利用しています。

## 公開リポジトリ

- [.github](https://github.com/AndanteTribe/.github) - 組織プロフィール用のリポジトリです。
- [Actions](https://github.com/AndanteTribe/Actions) - GitHub Actions 関連のコードをまとめています。
- [AssetsRegistry](https://github.com/AndanteTribe/AssetsRegistry) - Unity Addressables のアセットハンドルを一時管理するユーティリティです。
- [Backport](https://github.com/AndanteTribe/Backport) - Unity などの旧ランタイム向けに .NET API をバックポートしています。
- [Bindings](https://github.com/AndanteTribe/Bindings) - uGUI ベースの MVVM バインディングフレームワークです。
- [CommonUI](https://github.com/AndanteTribe/CommonUI) - 共通 UI 資産をまとめたリポジトリです。
- [cs-editorconfig](https://github.com/AndanteTribe/cs-editorconfig) - 共通の `.editorconfig` を提供します。
- [DebugToolkit](https://github.com/AndanteTribe/DebugToolkit) - ランタイムのデバッグメニューを作りやすくするライブラリです。
- [Extensions](https://github.com/AndanteTribe/Extensions) - 実用的な C# の拡張メソッド集です。
- [GameKernel](https://github.com/AndanteTribe/GameKernel) - エンジン非依存のゲーム開発向けドメイン基盤です。
- [GameObjectPool](https://github.com/AndanteTribe/GameObjectPool) - Unity 向けの非同期対応 GameObject プールです。
- [Inspector](https://github.com/AndanteTribe/Inspector) - Unity の標準インスペクターを拡張するカスタムインスペクターです。
- [Lengthy](https://github.com/AndanteTribe/Lengthy) - 長文表示に特化した Unity 向けテキスト UI ライブラリです。
- [Localization](https://github.com/AndanteTribe/Localization) - 書式対応のローカライズ支援ライブラリです。
- [LocalPrefs](https://github.com/AndanteTribe/LocalPrefs) - UnityEngine.PlayerPrefs の代替となるローカル永続化ライブラリです。
- [MasterSystem](https://github.com/AndanteTribe/MasterSystem) - .NET と Unity 向けのマスターデータ生成パイプラインです。
- [Memoria](https://github.com/AndanteTribe/Memoria) - PlayFab のランキングやスコアを扱うクライアント側 .NET 実装です。
- [ObjectReference](https://github.com/AndanteTribe/ObjectReference) - Unity オブジェクトの非同期読み込みを扱うための仕組みです。
- [Radio](https://github.com/AndanteTribe/Radio) - BGM・効果音・インタラクティブ音楽の再生を扱う Unity 向け機能です。
- [StandardUI](https://github.com/AndanteTribe/StandardUI) - Unity 向けの追加 uGUI コンポーネントとエフェクトです。
- [TapEffect](https://github.com/AndanteTribe/TapEffect) - Unity UI（uGUI）向けのタップ演出エフェクトです。
- [Toast](https://github.com/AndanteTribe/Toast) - Unity 向けのシンプルなトースト通知 UI です。
- [Transition](https://github.com/AndanteTribe/Transition) - Addressables ベースのシーン遷移システムです。
- [UniTaskPlus](https://github.com/AndanteTribe/UniTaskPlus) - UniTask の追加ユーティリティと拡張です。
- [UnityExtensions](https://github.com/AndanteTribe/UnityExtensions) - Unity の主要コンポーネントや型に対する拡張メソッド集です。
- [UnityTemplate](https://github.com/AndanteTribe/UnityTemplate) - Unity のテンプレートリポジトリです。
- [VContainerPlus](https://github.com/AndanteTribe/VContainerPlus) - VContainer の追加ユーティリティと拡張です。
